
# 足し算
![test](https://github.com/Takuto2345/robosys202x/actions/workflows/test.yml/badge.svg)

## 概要
標準入力された数の和を返すコマンド
## 必要なソフトウェア
* pyhon3  3.7, 3.8, 3.9, 3.10

## 使い方
Ubuntuをインストール後、pyhon3をインストールし、このリポジトリをダウンロードすると中に./plusがあるので
$seq ＜x＞ | ./plus 
を実行することで、和の答えが出力される。

## インストール方法
Ubuntu :https://jp.ubuntu.com/download 

## 仕様

## 例

```
$ seq 5 | ./plus
15
```
